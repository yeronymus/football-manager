# ⚽ Football Manager Bot

Автоматизированная система для организации любительских футбольных матчей. Бот берет на себя рутину капитана: сбор игроков, smart-балансировку составов, учет статистики и MVP.

> 🌟 **DevOps Showcase**: Этот проект служит демонстрацией инженерных практик (CI/CD, Monitoring, Dockerization). Архитектурные решения описаны ниже.

## 🚀 Live Demo & Доступ

Бот запущен на приватном сервере для локального футбольного сообщества. 
Так как бот управляет реальными играми и финансовыми ставками, публичный доступ к работающему инстансу ограничен.

*   **Хотите протестировать?** Свяжитесь со мной (контакты ниже), я выдам тестовый доступ или подниму демо-стенд.
*   **Хотите запустить у себя?** Инструкции по развертыванию (Docker Compose / K8s) есть в репозитории.

## 🛡 Архитектура и Безопасность

Система спроектирована с учетом защиты данных:
*   **Привязка к инстансу**: Бот может быть настроен на работу только с определенным администратором (`SYSTEM_OWNER_ID`).
*   **Защита API**: WebApp взаимодействует с бэкендом через валидацию `initData` от Telegram.
*   **Изоляция**: БД скрыта в внутренней сети Docker/K8s.

## 🛠 Технический стек

*   **Logic**: Python 3.11, Aiogram 3, SQLAlchemy (Async)
*   **Web**: FastAPI, Vanilla JS (WebApp)
*   **Data**: PostgreSQL, Redis
*   **Ops**: Docker Registry, GitHub Actions (Planning), Prometheus/Grafana (Planning)
