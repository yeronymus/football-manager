# Football Manager Bot (NSS)
Semestrální projekt pro předmět NSS. Jedná se o systém pro správu amatérských fotbalových zápasů (organizace, ELO rating, vyvažování týmů).
- **Autor:** Yernur Bauyrzhanuly, bauyryer
- **Architektura:** Modulární monolit 
- **Použité technologie:** REST API (FastAPI) v Pythonu, PostgreSQL, Redis (Cache/Messaging), Docker
