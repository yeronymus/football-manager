# ⚽ Football Manager Bot

Телеграм-бот для организации любительских футбольных матчей. Умеет собирать игроков, делить на команды по рейтингу (ELO), вести статистику и историю матчей.

## 🚀 Быстрый старт (Docker)

Самый простой способ запустить бота — использовать Docker Compose.

1.  **Клонируйте репозиторий:**
    ```bash
    git clone https://github.com/your-username/football-bot.git
    cd football-bot
    ```

2.  **Настройте окружение:**
    Создайте файл `.env` на основе примера и заполните его:
    ```bash
    BOT_TOKEN=ваш_токен_от_botfather
    ADMIN_IDS=[123456789]
    WEBHOOK_URL=https://ваш-домен.com
    POSTGRES_USER=postgres
    POSTGRES_PASSWORD=secret
    POSTGRES_DB=football
    POSTGRES_HOST=db
    POSTGRES_PORT=5432
    REDIS_HOST=redis
    REDIS_PORT=6379
    ```

3.  **Запустите:**
    ```bash
    docker-compose up -d --build
    ```

Бот запустится и будет доступен. База данных и Redis поднимутся автоматически.

## 🛠 Ручная установка

Если хотите запустить без Docker:

1.  **Установите зависимости:**
    ```bash
    python3 -m venv venv
    source venv/bin/activate
    pip install -r requirements.txt
    ```

2.  **Запустите базу данных и Redis** (локально или где-то еще).

3.  **Запустите бота:**
    ```bash
    python -m app.bot.main
    ```

## 🌟 Основные возможности

*   📅 **Создание игр**: Админ создает игру, бот пинит сообщение.
*   📝 **Запись**: Игроки записываются кнопками (+ / -).
*   ⚖️ **Баланс**: Бот делит составы так, чтобы силы были равны (Smart ELO).
*   📊 **Статистика**: Учет голов, побед, рейтинга.
*   📜 **История**: Архив матчей конкретного чата и личная статистика игрока.
*   🔔 **Напоминания**: Бот напомнит админу закрыть матч и внести счет.

## ⚙️ Настройки (config.py)

*   `SHOW_RATING`: `True` — показывать цифры рейтинга всем, `False` — скрытый режим (видит только админ).
