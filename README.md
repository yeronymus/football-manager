# ⚽ Football Manager Bot (Personal Edition)

Это **приватная система** для организации футбольных матчей, созданная для персонального использования. Бот автоматизирует рутину капитана: сбор игроков, деление на команды по скиллу (Smart ELO), ведение статистики и голосований.

## 🛡 Безопасность и Приватность ("Иммобилайзер")

Эта версия бота оснащена системой защиты **"Instance Locking"**:
*   **Привязка к владельцу**: Бот реагирует на команды *только* от владельца (`SYSTEM_OWNER_ID`).
*   **Защита API**: Все запросы к WebApp проверяются криптографически. Чужие пользователи получат `403 Forbidden`.
*   **Изоляция данных**: База данных хранится на зашифрованном сервере внутри Docker-контейнера.
*   **HTTPS**: Весь трафик шифруется через SSL (LetsEncrypt).

## 🚀 Основные возможности

### Для Организатора
*   **Создание игры**: Одной командой `/create`. Бот сам создаст опрос, закрепит сообщение и будет следить за записью.
*   **Умный Баланс**: Встроенный WebApp позволяет раскидать игроков по командам перетаскиванием (Drag & Drop) или нажать кнопку **"Авто-баланс"**, чтобы алгоритм разделил составы по рейтингу ELO.
*   **Учет Вратарей**: Отдельные слоты для GK с возможностью авто-распределения.
*   **Геймификация**: Рейтинг игроков пересчитывается после каждого матча.

### Для Игроков
*   **Простая запись**: Кнопки "+" и "-".
*   **Личный профиль**: Статистика игр, побед, голов и текущий рейтинг.
*   **Уведомления**: Бот тегнет только тех, кто записался, когда составы будут готовы.

## 🛠 Технический стек

Проект построен на современном асинхронном стеке Python:
*   **Ядро**: Python 3.11 + Aiogram 3.x
*   **API**: FastAPI (для WebApp и балансировщика)
*   **База данных**: PostgreSQL (Async SQLAlchemy 2.0)
*   **Кэш**: Redis (хранение состояний и прав админов)
*   **Инфраструктура**: Docker Compose + Nginx (Reverse Proxy)

## 📦 Установка и Обновление

Проект развернут на личном сервере.
Подробная инструкция по деплою и обновлению находится в файле:
📄 [DEPLOYMENT.md](deployment.md)

### Быстрые команды (для владельца)

**Обновить код на сервере:**
```bash
# С локальной машины
rsync -av --exclude 'venv' --exclude '__pycache__' --exclude '.git' --exclude 'football.db' . ubuntu@yernur-vm1.sin.cvut.cz:~/football_bot
ssh ubuntu@yernur-vm1.sin.cvut.cz "cd ~/football_bot && docker compose up -d --build"
```

**Остановить бота:**
```bash
ssh ubuntu@yernur-vm1.sin.cvut.cz "cd ~/football_bot && docker compose down"
```
